# Bank Management System (C++)

A simple console-based bank management system written in C++.

---

## Features
- Show all clients
- Add new client
- Delete client
- Update client information
- Find client by account number
- Save client data to a text file

---

## Data Storage
Client data is stored in a text file using this format:

AccountNumber#//#PinCode#//#Name#//#Phone#//#Balance


---

## Menu
1 - Show Clients
2 - Add Client
3 - Delete Client
4 - Update Client
5 - Find Client
6 - Exit


---

## How to Run
1. Open the project in any C++ compiler
2. Make sure `ClientsFile.txt` exists
3. Run the program

---

## Technologies
- C++
- File Handling
- STL (vector, string, fstream)
